Modules 06 / total

00foundation
graph-core core
Foundational graph database wrapper around velr — Db open/run/query, Cell / Table accessors, and Cypher string-escaping helpers used by every other module.
01engine
indexer-pipeline lib
Orchestrates the multi-phase indexing pipeline — walks the workspace, runs per-phase passes, and writes :File / :Symbol / :Package nodes into the graph.
01engine
lsp-adapter adapter
Language-server adapter inside the indexer — owns the persistent LspPool, drives per-language servers, and ingests their symbols into the graph.
02runtime
watcher infra
Filesystem-watch layer in the MCP crate — debounces notify events and re-drives the indexer pipeline in live mode, keeping the graph in sync with edits.
03surface
mcp-server protocol
MCP protocol surface — per-tool handlers (explore, find, notes, views, concepts, worklog, watch, report) plus shared util / tx wiring exposed to Claude Code.
03surface
indexer-cli cli
Standalone indexer binaries (bdd_viz and friends) — a thin CLI front-door over the same pipeline used by the MCP server.
